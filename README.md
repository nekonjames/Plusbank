# Plusbank
Plusbank is a demo bank account management REST service.

API Documentation is available here:
https://documenter.getpostman.com/view/6035213/RzfdqWLJ


What the task does: Create demo bank account, edit account name, credit bank account, delete account, transfer money between account, 
implement currency exchange rate conversion for transfer which source and destination currency are not same, load transactions history
and order by transaction date.
